# AI File Repair - Disk Data Recovery Tool

## 📖 重構說明

### 重新定位

**原本：** 文件修復工具
**現在：** 磁碟資料recover工具

### 核心功能

```
┌────────────────────────────────────────────────────────────────┐
│                    主要功能                                     │
├────────────────────────────────────────────────────────────────┤
│                                                                 │
│  🔍 磁碟掃描                                                     │
│  • 跳過 MBR/分區表                                              │
│  • 直接讀取磁碟 sectors                                         │
│  • 識別文件簽名 (Magic Numbers)                                 │
│                                                                 │
│  💾 資料recover                                                │
│  • 提取文件數據                                                 │
│  • 修復損壞的文件                                               │

│  🛠️ 修復功能                                                    │

│  • 批量修復                                                     │
│                                                                 │
└────────────────────────────────────────────────────────────────┘
```

### 使用流程

```
1. 運行程序
   ↓
2. 選擇磁碟 (跳過 MBR)
   ↓
3. 掃描並recover文件
   ↓
4. 修復recover的文件
   ↓
5. 保存結果
```

### 與 FinalData 的相似之處

| 功能 | FinalData | 本工具 |
|------|-----------|--------|
| 跳過 MBR | ✅ | ✅ |
| Raw Sector Reading | ✅ | ✅ |
| File Signature Detection | ✅ | ✅ |
| Data Recovery | ✅ | ✅ |

### 文件修復是附帶功能

```
主要：recover 出文件
次要：修復 recover 出的文件

如果磁碟正常 → 文件修復是主要功能
如果磁碟損壞 → recover 是主要功能
```

## 🔧 技術細節

### 跳過 MBR

```
直接訪問: \\.\PhysicalDrive0
→ 讀取 raw sectors
→ 不經過文件系統
→ 不依賴 MBR/分區表
```

### 文件識別

```
Magic Numbers:
• JPG: \xFF\xD8\xFF
• PNG: \x89PNG\r\n\x1a\n
• PDF: %PDF
• ZIP: PK\x03\x04
• MP3: \xFF\xFB
• MP4: \x00\x00\x00\x1f\x66\x74\x79\x70
...
```

### 修復流程

```
recover 文件
    ↓
檢查文件完整性
    ↓
AI 分析損壞類型
    ↓
選擇修復策略
    ↓
應用修復
    ↓
保存修復結果
```

---

## ⚠️ 注意事項

1. **需要管理員權限** - 訪問物理磁碟需要 elevated permissions
2. **只讀模式** - 預設只讀，避免進一步損壞
3. **備份優先** - recover 後請立即備份
4. **不要寫入原磁碟** - 避免覆蓋 recover 的數據
